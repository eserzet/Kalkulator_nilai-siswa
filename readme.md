## âœ¨ Fitur Unggulan  
- Input nama siswa dan pilih kelas (X, XI, XII)  
- Memasukkan nilai 12 (sekarang 13 jika kamu sudah tambah) mata pelajaran secara otomatis:  
  - Matematika  
  - Bahasa Indonesia  
  - Bahasa Inggris  
  - Fisika  
  - Biologi  
  - Kimia  
  - Geografi  
  - Ekonomi  
  - Sosiologi  
  - Pendidikan Agama  
  - Pancasila  
  - Seni Budaya  
  - **Informatika / Algoritma** (tambahan)  
- Validasi real-time nilai (harus 0-100)  
- Perhitungan: total nilai, rata-rata, predikat (A/B/C/D), status kelulusan (Lulus/Tidak Lulus)  
- UI menarik: animasi muncul, tema gradien, responsif di perangkat mobile  
- Fitur cetak dan download PDF hasil laporan (menggunakan jsPDF)

## âœ… Cara Menggunakan  
1. Buka aplikasi di browser lewat link live demo atau kloning repository ini.  
2. Masukkan **Nama Siswa** dan pilih **Kelas**.  
3. Isi nilai tiap mata pelajaran (0-100).  
4. Tekan tombol **Hitung** untuk melihat hasil: rata-rata, total, predikat, dan status.  
5. Untuk mencetak hasil atau menyimpannya sebagai PDF, gunakan tombol **Cetak** atau **Download PDF**.  
6. Untuk mengisi ulang dari awal, tekan tombol **Reset**.
